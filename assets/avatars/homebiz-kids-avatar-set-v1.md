# HomeBiz Kids Avatar Set v1

Purpose: playful, parent-approved illustrated child avatars for the HomeBiz Kids child portal. These are original vector-style concepts inspired by the direction Santiago liked, but without watermark/licensing risk.

## Style Direction
- Premium playful vector bust portraits
- Rounded shapes, soft shadows, optimistic expressions
- Kid-safe, cheerful, not corporate and not chore-tracker coded
- Consistent shoulders-up framing for app profile cards
- Warm off-white card background with teal/coral/yellow/lavender/blue/mint palette

## Honest Assessment
This set is directionally strong for product personality and licensing safety. The current SVG is a first-pass art-direction asset, not final polished brand art. Before shipping in-app, the best next move is to refine into individual transparent SVG/PNG avatar assets with cleaner line consistency and maybe two expression variants per avatar.
